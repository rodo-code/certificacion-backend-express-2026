## Instrucciones

Todo esto en su repositorio personal

0. Actualizar su .gitignore con el disponible en el repositorio del docente
1. Crear un directorio/carpeta en su repositorio
2. Hacer npm init en ese directorio
    entry point: main.js
    author: Su Nombre
    type: module
3. Instalar la libreria `random-item` con `npm install random-item`
4. Crear `main.js` con un simple `console.log` que imprima `Greetings [nombre] from Node.js`
5. Crear un script en `package.json` que corra `main.js` usando
`"start": "node main.js",`
6. Correr main desde node con `npm run start`
7. Hacer push en sus repos
